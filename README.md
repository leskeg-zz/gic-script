# gic-script
Script to get information from Exchange Service SOAP WSDL.

Setup
<br />
pip install -r req
<br /><br />
Execution
<br />
python main.py (to get data from last day)
<br />
python plot.py (to get and plot data every 30 minutes)
<br />
python demo_plot.py (to plot random data quickly)
